The Angular SPA has not been built yet.

To build and serve the Angular app:

  1. Navigate to C:\Projects\eShopOnContainers\src\Web\WebSPA\Client
  2. Run npm install
  3. Run npm run build:prod
  4. Copy the contents of dist/ to web/webspa-host/src/main/resources/static/
  5. Rebuild the Spring Boot jar and restart.
The Spring Boot host is running correctly at port 5104.
